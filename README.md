# 📊 E-commerce Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes e-commerce sales data to uncover key business insights related to revenue trends, product performance, and regional sales distribution. An interactive dashboard was built to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales performance across categories and regions
- Identify top-performing products
- Understand monthly sales trends
- Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Jupyter Notebook

---

## 📂 Dataset
- Superstore Sales Dataset (Kaggle)
- Includes order details, customer information, product categories, and sales data

---

## 🔧 Data Processing
- Cleaned dataset by handling missing values and duplicates
- Converted date columns into proper datetime format
- Created additional features like month for trend analysis

---

## 📊 Dashboard Features
- Total Sales KPI
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top 10 Products by Sales
- Interactive slicers for Region and Category

---

## 📈 Key Insights
- Technology category contributes the highest sales
- West region generates the maximum revenue
- Sales show seasonal trends across months
- A small number of products contribute significantly to total sales
